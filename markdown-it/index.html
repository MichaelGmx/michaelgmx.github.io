# 前言

查看 package.json 发现 使用的这个主题 shokax 没有使用 hexo 默认的 markdown 渲染（ hexo-renderer-markdown ），而是使用了自己定制的 hexo-renderer-multi-next-markdown-it 。

这不是关键，待我找到 hexo-renderer-multi-next-markdown-it 的 Github，
发现 作者又是对另一个 hexo-renderer-multi-markdown-it 做了定制。

还没有结束， hexo-renderer-multi-markdown-it 又又又又是对 markdown-it 做了定制。
markdown-it 其中包含 我这次真正想用的 markdown-it-container

只能说大佬们玩得太花，无限套娃。😹
