# 2. Hexo 问题 & 解决

# 2.1 hexo 引用本地图片无法显示

1. 安装 hexo-asset-image 插件
2. 修改插件中 code (参考上面 Url 中的指示)
3. 图片链接使用

hexo7.x 版本修改了渲染方式 hexo-asset-image 插件已归档 (长时间未维护)，所以不能使用了

替换方案为：

  1. 添加图片至 /source/_data/assets/文章名/图片.后缀
  2. 在文章中引用图片
